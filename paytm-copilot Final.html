Unknown UPI Fraud Detection

As you type the payment amount, Copilot changes color to signal risk — green for safe, orange for caution, red for danger.

  • 1
    Enter a UPI ID — Copilot checks if it's known/unknown
  • 2
    Type amount → color changes in real time
  • 3

  • 4
